# Simple Interest Calculator

This project is a simple Bash script to calculate Simple Interest.

## Usage
Run the script and enter Principal, Rate, and Time.

## Example
Enter Principal: 1000
Enter Rate: 5
Enter Time: 2
Simple Interest is: 100.0
